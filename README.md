# ScholarHub - Departmental Digital Resource & Knowledge Hub

ScholarHub is a premium departmental digital hub designed for seamless resource sharing, knowledge management, and community engagement.

## Features
- **Real-time Stats Dashboard**: Track resources, submissions, and completion rates.
* **Smart Repository**: Filterable access to departmental notes and textbooks.
- **Notice Board**: Instant broadcasts for departmental announcements.
- **Faculty Approval Workflow**: Robust verification system for student contributions.
- **My Progress**: Interactive activity visualization and ranking system.

## Tech Stack
- **Frontend**: React.js, Tailwind CSS (or premium custom CSS), Lucide React, Framer Motion, Recharts.
- **Backend**: Node.js, Express.js.
- **Database**: MySQL.

## Setup
### Backend
1. `cd backend`
2. `npm install`
3. Configure `.env` with your DB credentials.
4. `npm run dev`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm start`
